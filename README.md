# TO-DO
Developed a TO-DO Application using HTML,CSS and Javascript

# Technologies Used
- HTML for page structure
- CSS for page design
- JavaScript for adding, deleting, and completing tasks
- Local Storage for saving tasks in the browser

# Features
- Add a new task
- Mark task as completed
- Undo completed task
- Delete task
- Clear all tasks
- Show total, completed, and pending task count
- Tasks remain saved after refreshing the page

# How to Run
Open index.html in any web browser.
